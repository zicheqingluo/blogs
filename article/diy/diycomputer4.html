    # CPU降压

    B系主板搭载13系cpu普遍给的电压过高，如b760-g小吹雪d5+13700k ，默认电压给到1.47v，烤机最高功耗软显280w稳定下来也在250w左右。如果搭载z系主板则会动态调整电压，b系需要手动调整。

    测试平台：华硕B760-G小吹雪d5 13700k vk c360冷排

    # 教程

    # 更新bios

    • 进入华硕官网 (opens new window)-右上角服务支持-下载中心，选择主板型号，点击驱动程序和工具软件

    # ME

    • 更新ME版本,同bios，在bios最下面

    • 选择104微码

      • 关闭APE
      • AI Tweaker-Tweakers Paradise-Switch Mirocode:默认为CurrentMicroccode改为0下04Microcode
      • 重启电脑

    # 调整电压

    • AI Tweaker-Global Core SVIDD Voltage 改为Offset Mode
      • 便宜符号模式改为-
      • offset Voltage 改为0.1(可以继续加大找到最合适位置，本次为0.178)
    • AI Tweaker-Cache SVID Voltage 改为Offset Mode
      • 便宜符号模式改为-
      • offset Voltage 改为0.1(可以继续加大找到最合适位置，本次为0.178)

    # 防掉压

    • AI Tweaker - DIGI+ VRM - cpu负载线转正改为level1(如果开了ape选level3)

    # 重点

    • 防掉压1 可以使温度更低，但稳定性更差，aid64内存测试容易蓝屏
    • 开防掉压1关闭aep，开防掉压3开启aep
    • r23和aid64的内存及烤机都要测试，tm5最好也要测测

    # 内存超频

    既然选了13代u和华硕760小吹雪主板，就必然要折腾内存，虽然提升不了特别大的性能，就是体验折腾的乐趣。

    # 内存频率

    # 超频7000

    主板配置

    • 开机按DEL进入BIOS，按F7进入高级设置
    • 关闭XMP：选择AI Tweaker-Ai智能超频由XMP改为自动
    • 内存频率：DDR5-7000
    • 继续下滑修改电压：AI Tweaker-处理器系统代理电压由自动改为Manual Mode
      • CPU System Agent Voltage override别名sa设置为1.15000
      • High DRAM Voltage Mode由自动改为开启
        • DRAM VDD 和VDDQ电压改为1.45000
    • 内存时序设置：
      • 首选时序由全部auto改为从上到下32 42 42 42 64 指令速率还是auto不变。这里760主板将内存RAS#预充电延时分为了read write，网上教程里的主板bios部分read write只有1个。
      • 第二时序由全部auto改为从上而下 8 8
      • 内存行刷新周期时间：
        • DRAM Refresh Interval由auto改为65535
        • 内存写恢复延时由auto改为21
      • 第三时序
        • tWRWR_sg由auto改为16
        • tWRRD_sg由auto改为64
        • tWRRD_dg由auto改为46

    参考

    • [1] 超频7000 (opens new window)

    # 超频7400

    测试结论

    稳定通过tm5 aid64内存测试

    主板配置

    • 开机按DEL进入BIOS，按F7进入高级设置
    • 关闭XMP：选择AI Tweaker-Ai智能超频由XMP改为自动
    • 内存频率：DDR5-7400
    • 继续下滑修改电压：AI Tweaker-处理器系统代理电压由自动改为Manual Mode
      • CPU System Agent Voltage override别名sa设置为1.21500
      • High DRAM Voltage Mode由自动改为开启
        • DRAM VDD改为1.44000 和VDDQ电压改为1.42000
    • AI Tweaker\Advanced Memory Voltages(默认参数都是自动)
      • IVR Transmltter VDDQ Voltage改为 1.37000
      • Memory Controller Voltage改为 1.41600
      • PMIC Voltages 改为Sync All PmIcs
      • DRAM VDD Voltage 改为1.44000
      • DRAM VDDQ Voltage 改为 1.42000
      • DRAM VPP Voltage 改为 1.80000
    • 内存时序设置：
      • 首选时序由全部auto改为从上到下34 44 44 44 58 指令速率改为2N。这里760主板将内存RAS#预充电延时分为了read write，网上教程里的主板bios部分read write只有1个。
      • 第二时序由全部auto改为从上而下 8 8
      • 内存行刷新周期时间：
        • DRAM REF Cycle Time 2改为460
        • DRAMREF Cycle Time Same Bank 改为4 00
        • DRAM Refresh Interval由auto改为65535
        • 内存写恢复延时由auto改为48
        • 内存读取到预充电时间改为12
        • 内存四项动作成功时间改为16
        • DRAM WRITE to READ Delay L 改为16
        • DRAM WRITE to READ Delay S 改为16
      • 第三时序
        • tRDRD_sg-Runtime 改为16
        • tWRWR_sg由auto改为16

    参考

    • [1] 小吹雪专用7400过测 (opens new window)
    • [2] 未测试 (opens new window)

    # 超频6800

    • 开机按DEL进入BIOS，按F7进入高级设置
    • 关闭XMP：选择AI Tweaker-Ai智能超频由XMP改为自动
    • 内存频率：DDR5-6800
    • Memory Controler:DRAM Frequency Ratio:1:2
    • 继续下滑修改电压：AI Tweaker-处理器系统代理电压由自动改为Manual Mode
      • CPU System Agent Voltage override别名sa设置为1.11000
      • High DRAM Voltage Mode由自动改为开启
        • DRAM VDD改为1.40000
        • DRAM VDDQ电压改为1.37000
    • cpu电压：AI Tweaker-Advanced Memory Voltages
      • cpu VDD2(华硕叫MCV)电压改为1.35000
      • cpu VDDQ(华硕叫IVR)电压改为1.37000
    • 内存时序设置：
      • 首选时序由全部auto改为从上到下32 40 40 40 70 。这里760主板将内存RAS#预充电延时分为了read write，网上教程里的主板bios部分read write只有1个。
      • 第二时序由全部auto改为从上而下 8 8
      • 内存行刷新周期时间(TRFC)：
        • DRAM REF Cycle Time 2(TRFC)改为480
        • DRAMREF Cycle Time Same Bank 改为300
        • DRAM Refresh Interval由auto改为65535
        • 内存写恢复延时由auto改为48
        • 内存读取到预充电时间改为12
        • 内存四项动作成功时间改为16
        • DRAM WRITE to READ Delay L 改为16
        • DRAM WRITE to READ Delay S 改为16
      • 第三时序
        • tRDRD_sg_Runtine:16
        • tRDRD_dg_Runtine:8
        • tRDWR_sg:20
        • tRDWR_dg:20
        • tWRWR_sg由auto改为16
        • tWRWR_dg 由auto改为8
        • tWRRD_sg:60
        • tWRRD_dg:48
        • tRDRD_dr:1
        • tRDRD_dd:1
        • tWRWR_dr 由auto改为1
        • tWRWR_dd 由auto改为1

    # 结论

    超频每次提升较小，6400->6800读写复制有25%的提升，6800->7400 有25%的提升，所以为了功耗及稳定，超导6800足矣。
